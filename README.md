## nothing to do，没事干，来摸鱼

### 简介

这是一个基于[webman](https://www.workerman.net/webman)的爬虫项目，非常适合php框架的初学者学习，亦可用作毕业设计。

````
php7.4
mysql
redis
Crontab定时任务
Event事件
GuzzleHttp网络请求
QueryList识别html
api接口
nginx反向代理
````

### 安装

```
$ git clone https://github.com/echo-LuGuang/nothing_to_do.git
```

```
$ composer install
```

```
修改.env
```

Linux：
```
$ php start.php start
```

Windows：
```
$ windows.bat
```
